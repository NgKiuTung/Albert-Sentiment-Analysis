# ALBERT 情感识别系统

基于 ALBERT 模型的中文文本情感识别系统，支持 7 种情感分类。

## 项目说明

本项目实现了一个中文文本情感识别系统，采用 ALBERT 预训练模型进行微调。

### 支持的情感类型

- 喜欢 (Like)
- 悲伤 (Sadness)
- 恐惧 (Fear)
- 愤怒 (Anger)
- 厌恶 (Disgust)
- 快乐 (Happiness)
- 惊讶 (Surprise)

## 分支说明

- `main`: 主分支（空分支，仅包含说明文档）
- `Dev`: 开发分支（包含完整项目代码）

请切换到 `Dev` 分支查看完整项目代码和使用说明。

```bash
git checkout Dev
```

## 技术栈

- Python 3.x
- PyTorch
- Transformers (HuggingFace)
- Flask (Web服务)
- ALBERT Chinese Small
